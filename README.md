# ESE & GATE ME Coach (Netlify)

Sleek, day-by-day planner + AI mentor for **UPSC ESE (ME)** and **GATE ME**.

## Tech
- **Vite + React + Tailwind** (fast, modern)
- **Netlify Functions** for secure AI calls via **OpenRouter**
- **LocalStorage** persistence
- **Recharts** trendlines

## Quick Start (Local)
```bash
npm i
npm run dev
